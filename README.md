# HoppingWitch

## これはなんですか
Akashicエンジンで作成したフラッピーバードクローンです。

## 実行方法
```sh
$ akashic-sandbox .
```

## ゲームルール
画面をタップすると魔女が飛び上がります。上手く障害物をかわしてください。進んだ距離が得点になります。
